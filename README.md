# 🔥 RoastMyCode – AI Code Roaster & Fixer

Paste your code and AI roasts it like a comedian — then actually fixes it.
The most fun way to improve your code quality.

## ✨ Features
- 🔥 Savage Roast Mode — brutal honest AI feedback
- 🧑‍🏫 Mentor Mode — gentle, constructive suggestions  
- 🏆 Leaderboard of most-roasted code snippets
- Smooth CSS animations & shareable roast cards

## 🛠️ Tech Stack
React.js · Node.js · Express.js · MongoDB · Gemini API · CSS Animations

## 🚀 Getting Started
git clone https://github.com/magarenikhil748-debo/roastmycode-ai
npm install
npm start
